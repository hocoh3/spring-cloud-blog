关于博客社区

平台介绍

博客社区是一个基于Spring Cloud微服务架构的多用户博客平台，致力于为用户提供一个高质量的技术分享和交流社区。

我们的目标是打造一个专业、开放、活跃的技术社区，让每一位开发者都能在这里分享知识、交流经验、共同成长。

功能特点

多用户支持

支持用户注册、登录、个人主页管理等功能

丰富的文章管理

支持文章发布、编辑、分类、标签管理等

互动交流

支持文章评论、点赞、分享等互动功能

实时通知

基于WebSocket的实时消息通知系统

全文搜索

强大的文章全文搜索功能

响应式设计

适配各种设备的响应式界面设计

技术栈

后端技术

  • Spring Cloud微服务架构
  • Spring Boot框架
  • Spring Security安全框架
  • Spring Data JPA/MyBatis-Plus
  • MySQL数据库
  • Redis缓存
  • WebSocket实时通信

前端技术

  • HTML5/CSS3
  • JavaScript (ES6+)
  • Vue.js框架
  • Axios HTTP客户端
  • 响应式设计

基础设施

  • Eureka服务注册发现
  • Spring Cloud Gateway
  • Feign服务间通信
  • JWT身份认证
  • Docker容器化
